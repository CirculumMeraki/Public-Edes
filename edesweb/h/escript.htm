gsScript
SINTAXIS
eScript( $File )
DESCRIPCION
Devuelve la trayectoria real del fichero '$File', dentro del motor se manejan tres directorios por defecto:
1.- Si el nombre no empieza por "/" el script estará dentro de ".../Aplicacion/d/".
2.- Si empieza por una "/" el script estará dentro de ".../Aplicacion/".
3.- Si empieza por dos "/" el script estará dentro de ".../Aplicacion.file/".
4.- Si empieza por "=" el script será paralelo al directorio principal de la palicación
PARAMETROS
EJEMPLO
eScript('empresa.edf')                   // .../Aplicacion/d/empresa.edf
eScript('em/agencia.edf')                // .../Aplicacion/d/em/agencia.edf
eScript('/http/publico.html')            // .../Aplicacion/http/publico.html
eScript('//documento.pdf')               // .../Aplicacion.file/documento.pdf
eScript('//laboral/documento.pdf')// .../Aplicacion.file/laboral/documento.pdf
