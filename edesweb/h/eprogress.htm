eProgress
SINTAXIS
eProgress( window, Realizado [, Titulo [, TextoEnBarra [, TextoDetalle ] ] ] )
DESCRIPCION
Se muestra una barra de progreso.
Tambien hay una función en php de la forma:
eProgress( $Realizado [, $Titulo [, $TextoEnBarra [, $TextoDetalle ] ] ] )
La variable "$Realizado" tiene que pasar por los valores "0" y terminará con el valor "100" terminando la ejecución, no hace falta poner "eEnd()".
PARAMETROS
 window  
 Realizado  % del proceso realizad, de 0 a 100. Tambien se puede enviar una cadena con el nº de proceso actual mas la barra de dividir mas el total de procesos ej: "3/7"
 Titulo  
 TextoEnBarra  Si ponemos el simbolo "%" nos pondrá su % mas el símbolo
 TextoDetalle
