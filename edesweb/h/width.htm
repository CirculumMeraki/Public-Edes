Width
SINTAXIS
[Width] L:F | Field [,Field2, Field3,...] | Width / = [ | MínimoAncho]
DESCRIPCION
Define el ancho de la columna del control o su etiqueta en el formulario.
PARAMETROS
L:F El primer parámetro indica qué estamos tratando, el campo (F) o su etiqueta (L). 
Field Lista de campos separados por "," o un "*" para definir todos.
Width Puede tener 3 valores:
- Ancho en píxels.
- El carácter "=" con una lista de campos pondrá el ancho mas desfavorable .
- Nombre de un campo para aliniarlos con respecto a este.
MínimoAncho  Solo en caso de poner el parámetro "Width" como "="
EJEMPLO
[Width] F | dt_alta | 200
La columna en donde se ubica el campo dt_alta tendrá una anchura de 200 píxels.
EJEMPLO
[Width] L | cd_auto,cd_prov,cd_coma,cd_muni,cd_distri | =
Pone el ancho mas desfavorable.
EJEMPLO
[Width] L | cd_auto,cd_prov,cd_coma,cd_muni,cd_distri | = | 150
Pone 150 si el ancho mas desfavorable es menor a 150.
EJEMPLO
[Width] L | cd_auto,cd_prov,cd_coma,cd_muni,cd_distri | direccion
Alinea los campos con el campo "direccion"
