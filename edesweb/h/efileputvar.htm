eFilePutVar
SINTAXIS
eFilePutVar( $File, $Array )
DESCRIPCION
Modifica el valor de variables de configuración. Retorna si ha grabado bien.
PARAMETROS
$File  Nombre del fichero de variables 
 $Array Matriz bidimensional con el nombre de la variable a modificar y su valor 
EJEMPLO
eFilePutVar('/_datos/config/desktop.ini', array(
array('$_PrintMarginTop',12),
array('$_PrintHeader','Empresa ...')
)
);
