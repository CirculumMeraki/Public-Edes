eSLAction
SINTAXIS
eSLAction( NombreSubLista/null, Action )
eSubList( Action )
DESCRIPCION
Las SubListas tienen las opciones de Alta, Baja y Modificación en un submenú oculto que e muestra pulsando botón derecho en la SubLista, si queremos además tener iconos de acceso directo a las acciones pondremos esta función en el icono para asignarle una determinada acción a una SubLista.
PARAMETROS
El parámetro acción puede tener los siguientes valores: Insert(i), Update(u), Delete(d), View(v) y File(f) para ver el contenido de un fichero.
