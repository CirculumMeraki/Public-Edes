InsertToSeek
SINTAXIS
[InsertToSeek] [LabelButton] | [ReqFilter]
DESCRIPCION
Esta etiqueta es solo para el grupo de fichas en modo alta y sirve para que con un grupo de fichas se pueda ejecutar una búsqueda.
PARAMETROS
LabelButton   Label del botón del submit, por defecto "Buscar" y se puede poner el tag "IMG" o al principio un icono del sistema con "[PkIcono]" como la etiqueta [Icon].  
ReqFilter  Si el usuario debe meter algún filtro, por defecto "false".
EJEMPLO
[InsertToSeek] Buscar | true
 
EJEMPLO
[InsertToSeek] [S] Buscar | true
