DBGateway
SINTAXIS
[DBGateway] Mode | ScriptPHP
DESCRIPCION
Esta etiqueta sólo tiene validez en archivos EDF simples (no en grupos de fichas).
Cuando solicitamos una búsqueda desde un formulario, una vez encontrado algún registro, se ejecutará el script PHP introducido.
El script de esta etiqueta tendrá validez sólo en caso de que se encuentre algún registro. El modo para ejecutar el EDF puede ser "s" para ejecutar el script de la etiqueta DBGateway llegando a este script la variable "$_SQL" con la sentencia sql sin ejecutar y el modo "q" que es igual pero lo que llega al script es el cursor, es decir despues de ejecutar la sentencia sql.
PARAMETROS
ScriptPHP: Se le puede pasar un parámetro a este php poniendo un "=" y luego la asignación, ejemplo:
[DBGateway] S | inf_productos.php?=$cd_fabrica=12
EJEMPLO

[Title] INFORMES DE PRODUCCION

[DBGateway] S | inf_produc.php
