DESCRIPCION
Este código se ejecutará justo antes de atacar a la base de datos, una vez hecho el submit.
Es importante a la hora de manipular datos antes de que sean grabados, para modificar el valor de un campo se ha de utilizar la variable asociativa "$_vF".
Esta etiqueta no se ejecuta en los modos "c", "b" ni "m".

Si en el modo "B" está definida la variable "$_CHECKDELETE" se hará un chequeo antes de borrar el registro.
PARAMETROS
NomEDF: En los ficheros de definición "LDF" tipo de edición SubSelect "Ss" admiten este segundo parámetro para añadir "options" al select al principio cuando es atacado desde determinado sitio por ejemplo:
[DBIni] * | muni.edf
echo 'eAddOption( "NomCampo", Array( Array( 12, "InnerText" ) ) );';
