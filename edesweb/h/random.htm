Random
SINTAXIS
[Random] FieldForm, FieldTable, Table [, FieldForm=FieldTable ]
DESCRIPCION
La opción de desarrollo de rellenar el formulario de forma aleatoria, cuando el dato es seleccionar un dato de otra tabla se define con esta etiqueta.
PARAMETROS
Formato del cuarto parámetro:
        [Random] cif, cif, empre
     [Random] cif, cif, empre, cd_cli=fieldForm
     [Random] cif, cif, empre, cd_cli=63
     [Random] cif, cif, empre, cd_cli="const1"
     [Random] cif, cif, empre, cd_cli='const2'
     [Random] cif, cif, empre, cd_cli="{$_SESSION["..."]}"
EJEMPLO
[Random] cd_postal, cd_postal, postal

[Field]
     Código·Postal | cd_postal | CP | T | 5 | | M | | |
Como el código postal no puede ser un numero aleatorio la etiqueta [Random] nos dice que el campo "cd_postal" (primer parámetro) hay que ponerle el valor del campo "cd_postal" de la tabla "postal" de un registros aleatorio de esa tabla.
