eCallSrvPost
SINTAXIS
eCallSrvPost( url, Dim [, window]  )
DESCRIPCION
Envía un post a la "url" mediante una matriz asociativa por el ICallSrv , si en el último parámetro ponemos window desde la url donde se hace el post se podrá referenciar la página desde la que se llamó.
PARAMETROS
 
EJEMPLO
eCallSrvPost( 'edes.php?E:myscript.php', { 'var1':Valor1, 'var2':Valor2 } );
 
EJEMPLO
eCallSrvPost( 'edes.php?M', { 'RelationFields':'cd_auto,cd_prov,cd_coma', 'Values':'08,16,04' } ); -> 'EXEFUNC':'NomUser', 'DB':'oracle', 'SCRIPT':'...'

[RelationFields] cd_auto, cd_prov{}, cd_coma{}
                    08  ,    16    ,    04
