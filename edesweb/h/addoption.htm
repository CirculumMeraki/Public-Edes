DESCRIPCION
En un control "SELECT" añade más opciones "option" al principio de la lista.
PARAMETROS
Field: Nombre de un campo que aparezca en la etiqueta [Fields] cuyo tipo de control (par?metro 4) sea "SV" (Virtual Select).
Options_List: Pares de datos compuestos de un valor y un texto en la siguiente forma:
Valor1, Texto1; Valor2, Texto2; ......; ValorN, TextoN
Si el "Valor" es igual a "~" pondrá una linea divisoria.

Si en "Options_List" se encuentra el nombre de una función, sin parámetros la ejecutará, teniendo que devolver esta una matriz bidimensional con el valor interno y el externo del select respectivamente.
Otras opciones son:
- Poner una sentencia sql, con uno o dos campos y en este caso el siguiente parámetro puede tener el valor "BLANK" para que genere un option vacío.
- Poner un ">" seguido del nombre de un fichero plano, este fichero tendrá la definición del select virtual en la linea cero, pudiendo poner comentarios en las siguientes lineas.
- Poner el parámetro de la función "eFileGetVar()":
