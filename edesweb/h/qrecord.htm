qRecord
SINTAXIS
qRecord( $sql [[, arrayNumeric=false, allRow=false] ] )
DESCRIPCION
Por defecto devuelve una matriz asociativa con el primer registro que encuentre, poniendo "arrayNumeric" a true la matriz será numérica, utiliza un puntero propio.
Si le pasamos el parámetro "allRow" como "true" devolverá una matriz de registros.
PARAMETROS
 
EJEMPLO
$fields = qRecord("select user_surname, user_name from gs_user where cd_gs_user='{$_User}'");
$Apellidos = $fields["user_surname"];
$Nombre = $fields["user_name"];

lisT($Apellidos, $Nombre) = qRecord("select user_surname, user_name from gs_user where cd_gs_user='{$_User}'", true);
