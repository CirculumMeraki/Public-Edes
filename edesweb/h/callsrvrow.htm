CallSrvRow
SINTAXIS
[CallSrvRow] url | NmFunction | TextProgress [ | TextEnd ]
DESCRIPCION
Genera una llamada a la función "NmFunction" por cada fila del listado con la "url", generando un proceso masivo
PARAMETROS
url url a ejecutar para obtener la información, a esta url se añadirá de forma automática el nombre del campo serial y su valor en cada fila. Si el listado es paginado hará el proceso para todos no solo para los registros visibles.
NmFunction Nombre de la función a llamar cuando se quiera desencadenar el proceso, normalmente será el onclick de un icono.
TextProgress Texto al mostrar mientras se va ejecutando cada linea.
TextEnd Si se indica este texto será el mensaje al descargar un zip con todos los documento, sino está no generará ningún zip y solo informará que el proceso ha terminado.
EJEMPLO
[CallSrvRow] edes.php?E:factura_pdf.php | uFuncion | Descargando factura | Grupo de facturas
[SubTitle] eAddButton("pdf", "Descargar todas las facturas", "uFuncion()");


// Fichero: factura_pdf.php
...
if( $_GET["_CALLSRVROW"]<>"" ){
     eCallSrvRow("../_tmp/php/{$cod_factu}.pdf");     // sele pasa el nombre del fichero a descargar
}
Descargará un zip con todos los ficheros indicados
EJEMPLO
[CallSrvRow] edes.php?E:email.php | uFuncion | Enviando email
[SubTitle] eAddButton("email", "Enviar todos los email", "uFuncion()");


// Fichero: email.php
...
if( $_GET["_CALLSRVROW"]<>"" ){
     eCallSrvRow();                                   // no se le pasa ningún parámetro al no tener que descargar nigún fichero
}
Irá mandado los email (no descargará nada)
