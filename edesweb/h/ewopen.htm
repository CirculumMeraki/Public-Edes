eWOpen
SINTAXIS
eWOpen( EspecialURL, Condicion, Mensaje, ConIframe )
DESCRIPCION
Abre una nueva ventana estándar del navegador de forma mas cómoda controlando y centrando la ventana, con posibilidad de que se habra solo si se cumple cierta condición y en su defecto se mostraría un error. 
PARAMETROS
EspecialURL:  Este parámetro contiene la configuración de la ventana, su nombre y la URL.
SRT[*Ancho,Alto] NomVentana, Url
SRT: Flag para indicar: Resizable, Scrollbars y Toolbar.
[-] Ventana solo con la línea de título superior
[#] Ventana sin explorador visible, fullscreen.
[*] Ventana por defecto del window
[=] Ventana con la línea de títulos y la de estatus
[>] Ventana con el 'toolbar' para poder avanzar y retroceder página.
[D] Ventana definida por el usuario.
[eSWOpen] Ventana del sistema se utiliza cuando desde el arbol de opciones se quiere ejecutar siempre en una subventana del sistema, ej:
[eSWOpen]http://....

Condicion: Condición a cumplir para que se habra la ventana.
Mensaje: Si se especifica condición y no se cumple se mostrará este mensaje.
ConIFrame: Los valores posibles son true o false.
Si se indica el valor true se abre una ventana con un IFrame y dentro de éste se llama a la URL, esto es para evitar que en el título mientras se está cargando la página se vea la dirección URL.
EJEMPLO
top.eWOpen( '[-200,200]NomVentana,URL' );
top.eWOpen( '[-200,200]NomVentana' );        //.Ventana en blanco
top.eWOpen( 'left=400,status=1,top=50,width=300,height=300[D]NomVen,http://...' );
