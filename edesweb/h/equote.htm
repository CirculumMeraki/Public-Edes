eQuote
SINTAXIS
eQuote( Cadena / Matriz )
DESCRIPCION
El motor permite tanto las comillas dobles como las sencillas en el contenido de los controles, cuando se coge información de una tabla y se va a asignar tanto en JavaScript como en HTML valores que pueden contener tanto comillas sencillas como las dobles deberemos delimitador la cadena con comillas dobles y tomar el valor con la función "eQuote()" o anteponer la barra invertida "\" delante de las comillas dobles.
Devolverá la cadena/matriz transformada.
PARAMETROS
EJEMPLO

qQuery( "select cd_postal,nm_postal from postal where cd_postal='{$tmp}'" );
$loca = sql_Array();
eQuote( $loca );
