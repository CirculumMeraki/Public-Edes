Columns
SINTAXIS
{Columns} { NºColumnas [ | Propiedades [ | Title [ | CódigoEnLinea ] ] ]
...
}
DESCRIPCION
Esta etiqueta sólo funciona dentro de  la etiqueta [Fields] y sirbe para tener un número distinto de columnas dentro de su ambito, el final se marca con una llave de cierre "}".
PARAMETROS
Propiedades:
Por defecto se justitica a la izquierda, si sequiere centrar poner el flag "center".
Si se quiere que ocupe el 100% de la ficha poner el flag "100%".
Si se quiere que tenga border poner el flag "border".
EJEMPLO
[Fields]
...
{Columns}{ 3 | 100% | TITULO DEL GRUPO | style='border:1 solid red'
...
}
...
