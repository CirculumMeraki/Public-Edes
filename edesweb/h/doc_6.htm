Requisitos Previos 
Para utlizar e-Des es necesario tener conocimientos previos de HTML y CSS  además del lenguaje de script de sevidor PHP y  recomendable tener amplios conocimientos de JAVASCRIPT.

Haber programado script para  enlazar con bases de datos además de conocimientos de SQL.

Conocimientos del Navegador Internet Explorer, sistema operativo Windows y Linux.

Recomendable tener conocimientos de redes y protocolos asi como cortafuegos y niveles de seguridad para la versión de Intranet sobre Internet.
