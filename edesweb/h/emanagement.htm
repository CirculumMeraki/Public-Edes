eManagement
SINTAXIS
eManagement( ListMode [,Type=W [, FuncUser ] ] )
DESCRIPCION
Desde un listado saldr? un men? para su gesti?n. Una vez pulsado una opción se queda memorizado hasta que vuelvas a pulsar ptra opción.
PARAMETROS
ListMode   Lista de Modos a ejecutar, la inicial del label de todas las opciones son únicas y están reservadas las inicales "ABCM" para los modos de gestión.
 Las iniciales de gestión con: "I" Alta, "D" Borrar, "V" Consultar y "U" Modificar.
 El guión medio "-" generará una linea de separación.
 El guión bajo para las opciones del usuario es para que pregunte por un registro del listado
Type  Por defecto el submen? es de tipo ventana Type=W, pero con el Type='D' se colocar? abaja en una sola linea.
FuncUser  Funci?n de usuario al llamarlo en un modo personalizado

function uFunc( Op, Mode ){     // [Op]Opcion, [Mode]
