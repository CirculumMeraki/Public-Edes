eIconsShow
SINTAXIS
eIconsShow( [...] )
DESCRIPCION
Esta función sirve para pintar los iconos que se van a mostrar en el desktop, estos iconos se mantienen en la opción "Iconos Desktop", pues cada usuario puede tener asociado un número distinto de iconos, además a cada icono se le asocia un "Mode" ( Inser, Update, Delete, etc ) pudiendo parar la intranet para solo consultas para todos los usuarios menos a algunas excepciones.
Todos los iconos iconos u objetos se pintan dentro de una tabla en una celda independiente, la "class" de esta tabla es "ICONS" y el parámetro se introduce dentro de la definición de tabla.
PARAMETROS
EJEMPLO
eIconsShow( 'style="padding-right:50"' );
