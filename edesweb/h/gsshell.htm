gsShell
- Si pulsamos en el texto del directorio de la barra superior nos moveremos a él excepto si es el directorio actual que hará un "ls -lv".
- Si marcamos el nombre de un subdirectorio entraremos en él.
- Si marcamos un nombre de archivo lo añadirá mas un espacio a la línea de comandos, para evitar teclear.
- Para subir ficheros al servidor lo haremos mediante el botón "Examinar...".
- Para descargar ficheros utilizaremos el comando "cp NomFile".
- Con solo "cd" bajaremos un nivel de subdirectorios
- Funciona "cd.." como en MsDos sin espacio de separación
- Si tecleamos el nombre de un fichero lo visualizaremos
- Si tecleamos el nombre de un SubDirectorio nos moveremos a él sin necesidad de precederlo por "cd "
- A través de la orden "man" se puede ver la descripción de las opciones.
- Si marcamos un nombre de archivo lo añadirá mas un espacio a la línea de comandos, para evitar teclear etc.
- Si marcamos el nombre de un subdirectorio entraremos en él.
- Si pulsamos en el texto del directorio de la barra superior nos moveremos a él excepto si es el directorio actual que hará un "ls -l".
- Los comandos "ftpput" y "ftpget" el parámetro puede ser un fichero, una lista de ficheros separados por coma o un nombre de fichero con patrón valiendo los símbolos "*" y "?".
- El comando "cat" admite tres parámetros, el primero es el fichero si solo ponemos este se mostrará este si la longitud es menor de 250kb, longitud máxima por defecto, el segundo parámetro es para indicar la longitud máxima a mostrar,  si es negativo el valor los mostrará desde el final y si se pone el tercer parámetro entonces el segundo indica desde donde se empieza a visualizar y el tercero la longitud total a visualizar.
- El comando "hex" admite dos parámetros el primero es el nombre del fichero y el segundo opcional es la longitud a visualizar, por defecto son 50kb de fichero que equivalen 250Kb en pantalla por el "dump", el segundo fichero será la longitud a visualizar desde el principio si el valor es positivo y desde el final si es negativo el valor.
- El comando "recursive [rm|ls] patron" ejecuta de forma recursiva el comando "rm" para borrar ficheros o el comando "ls" para mostrar la lista.
