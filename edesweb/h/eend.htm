eEnd
SINTAXIS
eEnd( [$TextError=""] )
DESCRIPCION
Es la forma correcta de terminar cualquier petición en PHP en lugar de usar 'exit' o 'die()', dado que esta función comprimirá la página y mostrará todas las salidas de la etiqueta [DEBUG] con el parámetro "DF", así como cerrar la base de datos (transacciones). En el caso de tener la opción de cachear páginas en el servidor es esta función quien las cachea. Si llamaramos a un PHP mediante el método javascript 'startDownload()' se tiene que generar la salida sin compresión por tanto deberemos salir con 'exit' o 'die()'.
PARAMETROS
En caso de llevar parámetro si limipiará la página y se mostrará el mensaje de error.
