Radio
SINTAXIS
[Radio] Field | Config  | Value1,Text1 [; Value2,Text2]...[; ValueN,TextN] | Title | [Long1] [,Long2] ..... [, LongN] [ | QuitaSelección ]
DESCRIPCION
Define el comportamiento de los <imput> tipo 'Radio'
PARAMETROS
 Field  Campo existente en la etiqueta [Fields].
 Config  Número máximo de columnas y una letra para indicar que se pinta primero el label o el fields, "L" para indicar Label o una "F" para indicar Fields.
 ValueAndText  Pares de parámetros separados por coma: valor que se enviará al servidor, una coma y el texto a mostrar ( Value1,Text1 ).
 Title  Título del FIELDSET.
 Long  Logitud de la columna de datos a mostrar en cada columna indicada en el parámetro MaxCol.
 QuitaSelección  Por defecto es "false", se utiliza para cuando se quiere poder quitar la selección en todos sus valores.
EJEMPLO
[Radio] color  | 2L | B,Black; Y,Yellow; G,Green | Selecciona un color 
[Radio] propio | 4F | C,Ciudadano; L,Ayuntamiento; O,Otra Administración; E,Otra Entidad
