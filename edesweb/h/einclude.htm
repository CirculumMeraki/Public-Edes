eInclude
SINTAXIS
eInclude( NomRecurso [, NomRecurso,... ] )
DESCRIPCION
Incluye:
Los recursos del motor, si NomRecurso va sin extensión, indicados en el script del usuario si ha entrado por "edes.php".
Y los script de usuario si NomRecurso tiene extensión. Recordar que si no empieza por "/" lo leerá a partir de "d" y si empieza por "/" desde la raíz de la aplicación.
PARAMETROS
 mysql Controlador DB de MySql 
 informix Controlador DB de Informix
 oracle Controlador DB de Oracle
 message Función para mostrar un mensaje único en la pantalla 
 markdown Función para convertir de markdown a html "eMarkDown($txt)"
 lib Funciones:
eAddMonth( $AyoMes, $AddDelNumMeses )
eNum2String( $xNumero, $NDeci=0, $Salida='L', $SufijoNumero='', $SufijoDecimal='' )
Clases
eColor:
Metodo: Tone( $color, $Porcentaje )

[path]Script.Extensión  Script del usuario                                                                  
EJEMPLO
eInclude( $_Sql );// Se include el controlador de DB por defecto.
eInclude( 'mysql', 'message' ); // Include el controlado de MySql y la función de mensaje.
eInclude( 'cliente.php' );   // Include del script situado en el directorio "d" de datos
