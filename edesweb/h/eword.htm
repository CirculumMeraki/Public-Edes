eWord
SINTAXIS
eWord( "Comando|Parametro|Parametro|..." )
DESCRIPCION
Función para la comunicación con el Word.
PARAMETROS
NEW, N Nuevo documento Excel, se crea de forma oculta. top.eWord("N|NomFile");
OPEN, O Abre un documento existente. top.eWord("O|NomFile");
VISIBLE, V Muestra (1/T/TRUE) u oculta (0/F/FALSE) el documento top.eWord("V|T");
SAVE, S Graba el documento top.eWord("S");
SAVEAS, SA Graba el documento top.eWord("SA|NomFile");
EXIT, E, QUIT, Q Cerrar el documento top.eWord("E");
FREE Libera la memoria usada. top.eWord("Free");
PRINT Imprime el documento actual. top.eWord("PRINT");
MARGIN top.eWord("MARGIN|top,right,bottom,left");
ORIENTATION top.eWord("ORIENTATION|NumColumn");
TEXT top.eWord("TEXT|Motor edes");
SAVEASHTML top.eWord("SAVEASHTML|NomFile");

INSERTFILE

 top.eWord("INSERTFILE|{dir}prueba.txt");
REPLACE top.eWord("REPLACE|#|123")
REPLACESTYLE, RS Reemplaza y cambia el estilo, SubComandos:
"B", "BOLD"
"S", "SIZE"
"C", "COLOR"
"I", "ITALIC"
"N", "NAME"
"U", "UNDERLINE"
"T", "TEXT"
"CLEAR"		 WE.eWord("RS|...Txt a buscar...|C|1048320|B|1|S|30");
WE.eWord("RS|...Txt a buscar...|U|1|N|Verdana");
PICTURE top.eWord("PICTURE|Img|Img");
JUMP, J Salto de página (Jump) top.eWord("J");
