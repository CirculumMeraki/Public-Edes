eGO
SINTAXIS
eGO( NomCampo [ , AvisarError=true ] )
DESCRIPCION
Obtiene el objeto al que pertenece el campo especificado. Abreviatura de getObject.
Si no se le manda el nombre del campo tomará el valor del campo tipo INPUT que esté mas a la izquierda.
PARAMETROS
NomCampo    Nombre Campo / Id del objeto
AvisarError Por defecto si no encuentra el objeto saldrá un mensaje de error, si este parámetro le ponemos a false devolverá null si no existe el objeto
