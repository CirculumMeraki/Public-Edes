eDelOption
SINTAXIS
eDelOption(NomField [, DimValor])
DESCRIPCION
Borra uno o mas elementos "option" de un "select".
PARAMETROS
NomField  Nombre del campo o matriz con los parámetros
 DimValor  Matriz con los valores a borrar, si solo es un valor se puede pasar tal cual
EJEMPLO
eDelOption('cd_prov', Array( '01', '02'));    // Borra mas de un valor a la vez
eDelOption('cd_prov', '02');                  // Borra un único valor
eDelOption([
     ['condicion', ['A', 'S', 'E']],
     ['cd_tipo', ['1', '~' ]],                // borra también la linea divisoria
     ['cd_tipo', '2']
]);
