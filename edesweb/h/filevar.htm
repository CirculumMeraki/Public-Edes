FileVar
SINTAXIS
[FileVar] NomFile
DESCRIPCION
Consulta o Modifica mediante un formulario un fichero plano con variables de configuración, solo está activo en los modos 'cR' y 'mR'.
Identifica dos etiquetas [Title] y [Help].
Mostrará en la ficha:
- Si al principio de linea encuentra un comentario "//" hará un "HR".
- Si al principio encuentra "//[Title/Help]" interpretará estas etiquetas.
- Si encuentra un comentario multilinea lo pasa.
- Si empieza por "$" editará la variable y le pondrá como label el comentario que halla al final despues de "//".
- Informará de los comandos PHP: if, else, switch, case, default.
- Si al principio del label encuentra un corchete "[" entenderá que es un select virtual.
- Si al principio del label encuentra "[*]" el tipo de edición será entrada tipo "Clave".
- Si al principio del label encuentra un "-" la variable será solo de lectura
- Si se utilizan comentarios multilinea tanto "/*" como "*/" tienen que estar al principio de la linea.
- Cualquier resto de instrucciones las omitirá.
- Al grabar solo grabará en la parte de la asignación de las variables que el "$" esté al principio de la linea.
- Si el valor del dato es un color del formato "#000000" tendrá una ayuda para la selección del color.
PARAMETROS
EJEMPLO
$Usuario = 'NomUsuario';   // Usuario
$ColorBase = 'A';          // [A,Azul;V,Verde] Color base
$DataBase = 'mysql';// [M,MySql;I;Informix] Base de datos
$ClaveDB = '';             // [*] Clave para recuperar la DB
