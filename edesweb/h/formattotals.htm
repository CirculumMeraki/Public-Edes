FormatTotals
SINTAXIS
[FormatTotals] FormatCol1 [, FormatCol2] ..... [, FormatColN]  [[ | ColSpan ]     [ iSubList ] [ | AlineaciónColPrimera]
DESCRIPCION
Formatea las columnas de la línea de totales de un listado.
Si no deseamos que formatee una columna determinada, dejaremos el parámetro (su hueco) vacío.
Esta etiqueta es complementaria de TotalsRows, si TotalsRows no estuviese, esta etiqueta será ignorada.
Esta etiqueta también es multilinea de código PHP ( igual que [Format] ) donde se puede escribir código para modificar la fila a imprimir, se está escribiendo dentro de una función que recibe una matriz por referencencia llamada "$_vF" para modificar el cursor en curso, facilitando el realizar cálculos entre columnas.
PARAMETROS
Opciones de formateo
FormatCol A Código HTML libre, precedido de una comilla simple o doble (si introducimos el símbolo "#" se sustituirá por el valor de su celda).
B Si ponemos una "M" pondrá separador de miles 
D Si ponemos un número del uno al nueve será el número de decimales que deseemos mostrar
E Si ponemos una "B" los valores numéricos a cero no se verán.
D Se permite mezclar las opciones BD y CD, por ejemplo M2 o m2.
G Si ponemos una "N" no pondrá el separador de miles
H Si se pone una función se le pasarán dos parámetros, el nº de columna y una matriz con el valor de las operaciones que se hayan hecho en la fila
 I Una constante encerrada entre comillas dobles o sencillas.
J Una función de usuario.
ColSpan Indica en la linea de totales el número de colspan de la primera columna
AlineaciónColPrimera En caso de poner una constante en la primera columna, esta se puede alinear (L,C,R)
EJEMPLO
[ColsOp] ,+
[FormatTotals] 'SUMA TOTAL'
La etiqueta [ColsOp] indica que en su columna dos se sume el contenido y la etiqueta [FormatTotals] indica que en la primera columna ponga la constante 'SUMA TOTAL'.
EJEMPLO
[ColsOp] S,,+,,+,,+,,+,,+,,+,,+,, | Total
[FormatTotals] ,'Total'
El "Total" de la etiqueta [ColsOp] es para ponerlo en los subgrupos y el " 'Total' " de [FormatTotals] es para ponerlo en el total de totales y se pone en la segunda posición porque la primera columna es para hacer los subtotales "S".
