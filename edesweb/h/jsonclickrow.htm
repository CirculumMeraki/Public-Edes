JSOnClickRow
SINTAXIS
[JSOnClickRow] Mode [ [ | NomDF/else,... ] | NumCol/NomCol, NumCol/NomCol,... ]
...
DESCRIPCION
Código JavaScript a ejecutar al pulsar en un listado, la intención es hacer algún proceso sin que el listado se valla, no es seleccionar una fila (JSSelRow) sino ejecutar algún proceso relacionado con la fila, por ejemplo un listado de artículos y quieres mas información de uno en una nueva ventana sin quitar el listado. El tercer parámetro indica las columnas que son afectadas, por defecto son todas, las columnas indicadas en este parámetro tendrán el css "class=JSOnClickRow", dentro de la etiqueta esta disponible la variable "ColNumber" que indica el numero de la columna que se ha pulsado y "RowNumber" el numero de la fila.
Esta etiqueta se anula si utilizamos la etiqueta [JSSelRow]

Se pasan las siguientes variables de la fila pulsada.
     _vF = Matriz asociativa con el valor de las celdas.
     _oF = Matriz asociativa con los objetos (TD).
     _oTR = Objeto TR.
     _oTD = Objeto TD.
     $+NombreVariable = obtienes el valor o le asignas otro valor.

Variables/funciones obsoletas:
      _Celda  = La celda pulsada.
      _Fila  = La fila pulsada.
     _ValCol  = Matriz asociativa con el valor de las celdas de la fila pulsada.
     _Columna = Matriz ordinal con el valor de las celdas de la fila pulsada.
     eGF() = Con esta función podemos cojer el valor de cualquier campo de la fila pulsada.
                  También estan activas las siguientes matrices:

 _aTR[ nCol ] = Valor de la celda

_ColName = Nombre del campo
