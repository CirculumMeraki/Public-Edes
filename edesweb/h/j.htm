{J} RefName
DESCRIPCION
Es una sub-etiqueta de [Fields].
Añadir código JavaScript en una fila de la rejilla del formulario.
Dentro de la etiqueta [Fields], en la definición de un campo, como etiqueta insertamos "{J}" seguido de un nombre de referencia (por ejemplo "myRef"). Después de la etiqueta [Fields] volvemos a insertar "[J]" seguido del mismo nombre de referencia y en las siguientes líneas el código JavaScript que necesitemos. Con éste método podemos instroducir código JavaScript a nuestro antojo en cualquier fila de la rejilla del formulario.
PARAMETROS
RefName: Nombre identificativo.
EJEMPLO

[Fields]
   Client Code | cd_cli | ...
   {J} myRef
   ...

[J] myRef
