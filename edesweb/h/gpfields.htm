GPFields
SINTAXIS
[GPFields] Fields | Caracteres
DESCRIPCION
Es una etiqueta para agilizar la prueba de formularios. Si tienes que probar el alta de un formulario de muchos campos en cada prueba tienes que introducir cada vez todos los datos, pues si pulsas el botón derecho en la página principal para sacar el menú de desarrollo tienes dos iconos uno de 'Copy' y otro de 'Paste' para copiar y reproducir los datos del formulario y para que puedas dar altas seguidas y los datos puedan variar para eso es esta etiqueta, si pones una lista de caracteres los generará de forma aleatoria en el campo y si pones un '+' pondrá un número secuencial, es decir primero defines mediante la etiqueta [GPFields] los campos que tienen que cambiar su valor al ser campos únicos ( ej el DNI, el CIF, etc ), despues rellenas un alta y antes de dar a submit haces "copy" con el menú del desktop o el de la pagina actual, haces submit y la proxima vez que salga la ficha de alta si pulsas a "Paste" se copiaran todos los valores de los campos del alta actual variando los valores de los campos indicados en las etiquetas [GPFields]..
PARAMETROS
Caracteres: Es una lista de caráctes que acepta el campo o un '+' para indicar campo autoincremental
EJEMPLO

[GPFields] codigo | +
[GPFields] nombre | A BCDEFGHIJKLMNOPQRSTUVWXYZ //Genera un nombre aleatorio

El primer ejemplo hará que cada vez que des a "paste" en el campo 'codigo' se ponga un número secuencial y en el segundo ejemplo pondrá de forma aleatoria los caracteres de la 'A' a la 'Z', fijaros que si queremos generar aleatoriamente caracteres en blanco este caracter le tendremos que poner dentro de la secuencia.
