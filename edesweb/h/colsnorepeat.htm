ColsNoRepeat
SINTAXIS
[ColsNoRepeat] ,...
 Sintaxis alternativa
[ColsNoRepeat] NomCampor [,...] [ | classNameTD [ NOEMPTY] ]
DESCRIPCION
En los listados dato a dejar en blanco si se repite en la fila anterior. Esta etiqueta solo tendrá efecto en html y pdf, si se pone el parámetro "classNameTD" le pondrá la clase para poder identificarlo.
En el segundo parámetro se puede poner las siguientes particulas del sistema:
     - NOEMPTY: Se utiliza para no dejar el datos en blanco y ponerle otro color.
     - TRANSPARENT: Clase del sistema para poner el color muy tenue.
PARAMETROS
 
EJEMPLO
[ColsNoRepeat] true,1
 
EJEMPLO
[ColsNoRepeat] cd_auto,cd_prov
 
EJEMPLO
[ColsNoRepeat] a, b, c, d | NOEMPTY TRANSPARENT
