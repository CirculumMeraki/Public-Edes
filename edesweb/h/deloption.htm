DelOption
SINTAXIS
[DelOption] Mode | Field | OptionsList [ | noEmpty ]
...
DESCRIPCION
El campo especificado debe ser un campo tipo "
PARAMETROS
Field: Campo que exista en la etiqueta [Fields].
OptionList:  Lista de valores separados por coma, si algún valor empieza por "$" se considerará una variables y se sustituirá por su valor.
noEmpty: Indica que el select no tenga el primer elemento vacío.
