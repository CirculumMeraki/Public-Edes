eRelationField
SINTAXIS
eRelationField( window, Matriz )
DESCRIPCION
Carga una relación de select con un solo acceso al servidor.
PARAMETROS
 window Constante
 Matriz Una matriz Hash con los siguientes valores:
RelationFields ':'cd_auto,cd_prov,cd_coma'
Values = Lista de valores separados por coma
EXEFUNC = Función a ejecutar al terminar la carga (opcional)
DB = Nombre del fichero de definición del DDBB alternativo (opcional)
SCRIPT = Nombre del script que devolverá mediante un "return" una matriz  con todas las submatrizes como select hay que rellenar (opcional)
EJEMPLO
top.eRelationField( window, { 'RelationFields':'cd_auto,cd_prov,cd_muni', 'Values':'08,16,004' } );
Cargará los select "cd_prov" y "cd_muni" y pondrá a los tres select los valores indicados.
EJEMPLO
top.eRelationField( window, { 'RelationFields':'cd_auto,cd_prov,cd_muni', 'Values':'08,16,004', 'Script':'cargaselect.php' } );

Script: "cargaselect.php"
<?PHP
...
$Dim = array(2);
$Dim[0] = Matriz de dos dimensiones con los datos de la provincias.
$Dim[1] = Matriz de dos dimensiones con los datos de los municipios.
...
return $Dim;
?>
Los select "cd_prov" y "cd_muni" se rellenan con la matriz "$Dim" devuelta por el script "cargaselect.php"
