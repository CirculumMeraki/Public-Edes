DESCRIPCION
Establece el máximo número de segundos que se le permite a un script ejecutarse, si se establece a cero, no se impone ningún límite. Esta etiqueta solo es necesaria si hay que cambiar el valor por defecto de PHP. 
PARAMETROS
EJEMPLO
[Expire] 180
Espera 180 segundos para caducar la página.
