PDFPhp
SINTAXIS
[PDFPhp] Type | NomFile
DESCRIPCION
Código PHP a incluir al generar salidas a PDF. Se utiliza para generar funciones para imprimir datos. Por defecto lo busca en el subdirectorio 'd'.
Y dentro del fichero donde está definida la ficha que sale en PDF habrá una llamada a alguna función de dentro del fichero ej. "E; uPintarFotos( 2, 15, 20, 0, 6 )"
PARAMETROS
Type : F : Unico valor actual.
EJEMPLO
[PDFPhp] F | info_fotos.php
