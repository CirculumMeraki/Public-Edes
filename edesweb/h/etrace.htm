eTrace
SINTAXIS
eTrace( $txt, $tipo=false )
eTrace( Array )
DESCRIPCION
Es una función de depuración y rastreo, solo se debe utilizar en tiempo de desarrollo. Muestra por pantalla o mediante un 'alert()' de javascript el contenido de la variable "$txt", si 'tipo' es true y se está en javascript el mensaje se mostrará en un 'alert()' y si es HMTL se mostrará en texto sin resaltar.
Cuando esta función se ejecuta e un proceso Background la información se grama en el fichero "/_tmp/err/_log.err".
El parámetro "$txt" puede ser una cadena o una matriz.
