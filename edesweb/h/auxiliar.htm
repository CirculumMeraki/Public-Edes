DESCRIPCION

Este fichero es para cargar select en formato de tabla o en formato select en el momento de la carga de la aplicacion y el resto de veces que se tenga que usar viajara vacio y al llegar lo copiara ( Ubicación: _datos/config/auxiliar.usu ).

Tabla auxiliar
eLoadList( $Tabla, $Titulo, $Where );
Ej: eLoadList( 'gs_node', 'Local', '' );

Tabla con niveles
eLoadTList( $Tabla, $Titulo, $Where );
Lista de campos: cd_[Tabla], nm_[Tabla], nivel, orden
Ej: eLoadTList( 'cnae', 'C.N.A.E', '' );

Select auxiliar
eLoadSelect( $NomSelect, $Tabla, $Campos, $Condicion, $Orden );
Ej: gsAuxSelect( 'autonomias', 'auto', 'cd_auto,nm_auto', '', 'nm_auto' );

Añade indices al select
MasSelect( $Campo, $Contenido );
Ej: MasSelect( 'nivel', '1,Básico; 2,Medio; 3,Total' );
