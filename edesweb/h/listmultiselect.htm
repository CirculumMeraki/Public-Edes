ListMultiSelect
SINTAXIS
[ListMultiSelect] Label | script.php / functionJavascript() [ | CheckRow [ | row ] ]
DESCRIPCION
Mostrará un listado pudiendo seleccionar cualquier número de filas, ejecutando despues un script php o una funcion javascript.
Si se ejecuta un script php le llegará un array en el campo $_POST["_PK_MULTISELECT_"].
Si se ejecuta una funcion javascript recibir? una matriz en el primer argumento.
PARAMETROS
Label   Label del botón a ejecutar al terminar la selección.
script.php / functionJavascript()   - script php
 - nombre de función javascript
(permite incruscar variables php)
CheckRow  Función javascript para confirmar/denegar que se pueda seleccionar la fila
row   Por defecto enviará los el campo indice de la tabla
 Constante para indicar que se enviarán los rowIndex y no el campo indice
Para marcar filas en la carga se hace mediante la etiqueta [Format]
[Format]
if( $_vF[...HayQueMarcar...] ){
   $_vF[..ColumnaDeLaMarca...] = true;// la última mas uno
   $_RowClass = "MarkedRow";
}
