eShowOption
SINTAXIS
eShowOption(  NomSelect, DimValores/Valor [, true/false ] )
DESCRIPCION
Oculta o muestra determinados valores de un select
PARAMETROS
 NomSelect  Nombre de campo select.
 DimValores/valor  Este parámetro puede ser un valor o una madrid de valores.
 true/false  Indica si el option será visible, pordefecto true.
EJEMPLO
eShowOption( 'cd_prov', Array( '01','02'), false );
eShowOption( 'cd_prov', '01', false );
