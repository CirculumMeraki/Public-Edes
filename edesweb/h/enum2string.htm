eNum2String
SINTAXIS
eNum2String( $Numero, $NumDecimales=0, $FormatoSalida='L', $SufijoEntero='', $SufijoDecimal='' )
DESCRIPCION
Te devolverá un número en formato texto. Para utilizar la función se ha de cargar mediante "eInclude('lib')".
PARAMETROS
$Numero Número a pasar a texto
$NumDecimales Número de decimales 
$FormatoSalida Formato salida: U.Upper, L.Lower, C.Capitalize
$SufijoEntero Sufijo a añadir después de la parte entera, se puede utilizar el símbolo "/" para construir el plural ( igual que en [Title] )
$SufijoDecimal Sufijo a añadir después de la parte decimal, se puede utilizar el símbolo "/" para construir el plural ( igual que en [Title] )
EJEMPLO
echo eNum2String( $Importe, 2, 'U', ' euro/s', ' centimo/s' );
