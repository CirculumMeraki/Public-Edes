WhereSelect
SINTAXIS
[WhereSelect] Mode | Field | SQLCondition
...
DESCRIPCION
Es la condición / filtro ( WHERE ) a ejecutar en el campo select y el "order by".
En el parámetro "SQLCondition" podremos poner una variable php de la forma "{$Var}", esta variable deberá ser pública para tener acceso a ella.
También se puede cambiar la ordenación del select añadiendo en "SQLCondition" el texto "desc", pudiendo contener este parámetro solo una condición, una condición mas la ordenación descendente o solo el tipo de ordenación.
En los listados no está activo pero sí en las zonas EditList.
PARAMETROS
Field: Nombre de un campo existente en la etiqueta [Fields].
SQLCondition: Condición SQL
EJEMPLO
[WhereSelect] *    | cd_tipovia   | cd_tipo='V'
[WhereSelect] mR,a | cd_propiedad | nm_propiedad[1,3] != "(*)"

[WhereSelect]
     * | cd_prov | desc
     * | cd_prov | nm_prov < 'D' desc
     c | cd_prov | nm_prov like '%I%' order by nm_prov desc
     c | cd_prov | order by nm_prov desc
