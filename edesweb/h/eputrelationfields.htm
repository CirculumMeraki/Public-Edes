ePutRelationFields
SINTAXIS
ePutRelationFields( NomField, Value [, ConOnchange=true ] )
DESCRIPCION
Los campos que están en un [RelationFields] se deben modificar desde javascript mediante esta función para que se ejecute el evento onchange.
PARAMETROS
EJEMPLO

ePutRelationFields('cd_auto','01');
ePutRelationFields('cd_prov','11');

Aunque el select "cd_prov" este vacío en el momento de la asignación se recargará y pondrá como valor por defecto el valor "11" y si tuviera valor interno los select también se asignarían.

EJEMPLO
ePutRelationFields('cd_auto','01');
ePutRelationFields('cd_prov','11',false);
ePutRelationFields('cd_muni','001');

Este es el caso de 3 select relacionados, fijarse que en el campo "cd_prov" se desactiva el "onchange" para que no se recarge el campo "cd_muni" dos veces.
