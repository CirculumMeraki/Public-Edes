¿Cómo funciona E-DES?
E-des es un entorno basado cien por cien en el navegador y su versión 3 exclusivamente en Internet explorer de Microsoft. 

Nos permite desde cualquier punto del planeta, diseñar un formulario, hacer una modificación de una tabla, cambiar un report,etc. Solo depende de las reglas impuestas por los cortafuegos de las Intranets de cada organización,cliente, empresa o usuario particular.

E-Des, como ya explicamos en el apartado, ¿Que es?, esta dividido en dos partes, el Entorno de Desarrollo y Motor de Ordenes.

El Entorno de Desarrollo se instala y configura siguiendo los pasos definidos, sobre el servidor web o estación de trabajo 
A continuación ejecutamos una pagina Html sobre el servidor de paginas que nos da acceso a configurar todo el sistema de Intranet que pretendemos montar, esta aplicación es gsMain. Los derechos sobre esta aplicación estan restringidos al Jefe de proyecto, Analista Jefe o responsable del proyecto, que debera activar usuarios y permisos.



Despues de terminar con la configuracion generaremos una maqueta del proyecto. A partir de ese momento activaremos el programa gsEdit desde la maqueta para inicia la programación de todos los formularios,listados,etc.

Finalizado el proyecto, generaremos el proyecto definitivo y será instalado en el servidor Web del Cliente final. El proyecto se compone de todos los ficheros de configuracion y el Motor de Ordenes.

Los desarrolladores podran generar versiones y parches del programa desde la maqueta de desarrollo y enviadas automaticamente al servidor web del cliente.
