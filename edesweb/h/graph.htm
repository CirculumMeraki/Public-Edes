[Graph]
SINTAXIS
[Graph]  [R/C/A] | ROW(y) | COL(x) | TERRITORIO(x) | ALL COL(x) | LegendTitle | Variable=Valor, ...
DESCRIPCION
Genera gráficas partiendo del listado.

$_GRAPHHIDECOL[5] = true;
$_GRAPHPERCENTAGE['C'] = 1;

Si existe la función eGraphUser() se llamará para modificar las variables por cada tipo de gráfica, se definiría de la siguiente forma:
function eGraphUser( $TipoGRafica, &$_Form, &$_COLSOP, &$TituloGrafica, &$TituloColumna, &$TituloFila, &$TituloLeyenda ){
...
}
También se puede llamar a la función "eGraphUserData" en los tipos Columns y Pie:
eGraphUserData( $TIPO, $_Form, $DatosCol ){
...
}
PARAMETROS
Type C: Gráfica de las columnas (Columns)
R: Gráfica de las filas (Rows)
G: Gráfica del cruze de columnas por filas (Grid)
Tipo de gráfica separada por coma o dejar en blanco y generará las tres.
P: Tarta (Pie)
RowTitle Título de las Filas 
ColTitle Título de las Columnas en Col
ColTitle Título de las Columnas en Row 
  Col x Row
LegendTitle Título de la Leyenda 
Variable=Valor

Lista de variables con su valor para cambiar la configuración,las variables a cambiar son:
Decimals

BarColorGradient
BarBorderColor
BarAxisColor 
BarLightPosition

GraphBorderColor
GraphLayerColor

TitleColor
TitleRowColor
TitleColColor

DataColor

GrillColorOne
GrillColorTwo

TitleFontType
FontType
Todas las variables menos Decimals se encuentran en el fichero de configuración "/_datos/config/graph.ini"

EJEMPLO
[Graph] * | VOTOS | CANDIDATOS | TERRITORIO | CANDIDATOS x TERRITORIO | CANDIDATOS | BarColorGradient='#FFFFFF', BarColor0='#00FF00'
