qQuery
SINTAXIS
qQuery( $Sql [ [, $Cursor] , $Continuar=false ] )
DESCRIPCION
Ejecuta una sentencia SQL y retorna el número de registros afectados, en el caso de "Informix" ese número es aproximado.
Si se ha insertado un nuevo registro se puede obtener su serial mediante la matriz asociativa $_IdRegistro[NmTabla].

Con el control de oracle en las inserciones se puede definir el tipo secuencia a utilizar mediante la variable $_DBSEQUENCE
1.- Por defecto el motor usa la secuencia "sq[NmTabla]"
2.- Si definimos $_DBSEQUENCE="NO" la secuencia la generará el trigger definido en la tabla.
3.- Si $_DBSEQUENCE<>"" la secuencia será el valor de la variable $_DBSEQUENCE
PARAMETROS
$Sql: Sentencia SQL a ejecutar.
$Cursor: Parámetro opcional para indicar sobre que cursor se quiere ejecutar, si se omite será sobre el cursor actual.
$Continuar: Indica si hay un error SQL /DB si se para el script o te deja continuar, en cuyo caso la función devolverá "-1".
