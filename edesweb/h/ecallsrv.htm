eCallSrv
SINTAXIS
eCallSrv( window, url [ , Condicion=true ]  )
DESCRIPCION
Conecta con el servidor de forma transparente. Suele usarse con la etiqueta [OnChange] para completar datos o chequearlos, la ejecución se hace mediante un iframe ubicado en el desktop. Si se quiere evitar que se ejecute en el inicio, cuando se pone algún dato por defecto, la función se llamará con un guión bajo delante "_eCallSrv()".
PARAMETROS
window: El objeto "window" de la ventana actual.
url: Nombre del script a ejecutar, si no tiene extensión se asumirá '.php'.
EJEMPLO
[OnChange] a | dni | eCallSrv( window, 'existe_dni.php?'+this.value )
