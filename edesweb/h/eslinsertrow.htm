eSLInsertRow
SINTAXIS
eSLInsertRow( Obj )
DESCRIPCION
Crea un nuevo TR en la sublista indicada devuelve el objeto TR de la sublista con las celdas creadas y vacias.
PARAMETROS
Obj Es el objeto table de la sublista, es el nombre de la sublista que se define en la etiqueta [Fields] 
EJEMPLO
[Fields]
   Campo1  | campo1     | X | T | 30 | | M | | | 
           | [SubList2] | o |   |    | |   | | |
...
var Obj = eSLInsertRow( document.getElementById('[SubList2]') );
Obj.cells[0].innerText = '....';
