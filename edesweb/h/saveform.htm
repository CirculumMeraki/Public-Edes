SaveForm
SINTAXIS
[SaveForm] Mode | NomFile
DESCRIPCION
Orden de desarrollo para grabar la parte del formulario a un fichero php externo para modificarlo y adaptarlo, cuando con eDes no se le puede dar la forma que se quiere. Las opciones "c", "b", "m" y "a" si no tiene valores por defecto se puede usar tal cual pero si utilizáramos la opción por ejemplo "mR" que contendría datos hay que sustituir los datos por variables PHP para que se adapte a cualquier situación. Esta etiqueta solo se usa en desarrollo para generar la base del fichero a modificar una vez generado se quitará y en su lugar se usara [ExtForm] para importar el formulario personalizado. Opción no disponible en el grupo de fichas.
