eEditField
SINTAXIS
eEditField( Campo, OnOff [ [ , CSS ], ImgOnOff ] )
DESCRIPCION
Activa/Desactiva la edición de un campo pudiendo cambiar su estilo, también se puede usar el alias "eEF()".
También tiene la función de ocultar/mostrar el botón de submit y los botones adicionales, así como activar y desactivar iconos.
PARAMETROS
 Campo  Campo o lista de campos separados por coma.
Si el campo es "OpEXE" ocultará/mostrará el botón de submit, también se puede usar la función eDisableButton().
Si el campo es "AddButton[NºBotón]" ocultará/mostrará el botón indicado.
Se pueden desactivar todos los campos a la vez con el asterisco eEF("*",false);
Si el id es de una imagen la podrá desactivar y activar para que se pueda ejecutar sus eventos.
 OnOff Valor lógico para indicar si el campo es editable true/false 1/0, solo actua sobre los campos visibles.
Si se quiere actuar sobre los visibles y los ocultos pasaremos los valores "-1" y "2".
 CSS Contenido opcional del atributo "Style"
ImgOnOff Valor lógico para indicar si el icono asociado está activo o no, por defecto no se toca.
