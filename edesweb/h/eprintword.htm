ePrintWord
SINTAXIS
ePrintWord( NomPlantilla, DimVariables, SgClose [, Setup, CopiarComo, GrabarCambios, DimJS ] )
DESCRIPCION
Imprime un documento MSWord de forma oculta sustituyendo el valor de variables dentro del documento.
PARAMETROS
 NomPlantilla Nombre completo de la plantilla ( fichero local ).
 DimVariables Matriz de dos dimensiones conteniendo el nombre de las variables a sustituir y su contenido.
 SgClose Segundos en cerrarse la instancia del Word, hay que dejarle el tiempo suficiente para que pueda imprimir y después a los segundos indicados se cerrará la instancia del Word, si el valor es menor de uno no imprimirá, si es "0" dejará la instancia visible y si es "-1" invisible y cerrará la instancia.
 Setup Cadena de valores de configuración separados por comas (si algún parametro no se quiere tocar sedejará vacia su posición) con la siguiente disposición: "PageWidth, PageHeight, TopMargin, RightMargin, BottomMargin, LeftMargin"
 CopiarComo Copia "NomPlantilla" con el nombre "CopiarComo".
 GrabarCambios  Los cambios se graban los cambios cuando está el parámetro "CopiarComo".
 DimJS Matriz con ordenes javascript a ejecutar antes de salir de la función. Ejemplos:
ActiveDocument.CheckSpelling();
ActiveDocument.CheckGrammar();
EJEMPLO

top.ePrintWord( 'c:/edes/doc/tiket.doc', Array(
Array("%%importe%%", uImporte),
Array("%%iva%%"    , '6,00'  ),
Array("%%total%%"  , uTotal  ) ),
10, true );
