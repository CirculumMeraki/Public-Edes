RelationFields
SINTAXIS
[RelationFields] FieldList [ [ [ | CampoASaltarse ]  | Saltar=true ]  [ | FieldListInMemory  ]
...
DESCRIPCION
Forma dependencias entre campos para utilizarlos en los "SELECT's" anidados. A parte de esta etiqueta en la etiqueta [Fields] hay que definir la lista de campos del tipo "Ss" SubSelect, el primer campo puede ser del tipo que se quiera, normalmente será un "S" Select o un "T" Texto.
Cuando se cargan los subselect se puede ir a una fucnión de usuario para por ejemplo eliminar algún option, ver ejemplo en la etiqueta [OnChange].
PARAMETROS
 FieldList Lista de campos separados por comas, en orden de dependencia.
 CampoASaltarse En caso de querer saltarse un select intermedio pondremos el nombre del campo a saltarse de forma opcional.
Si uno de los campos es el primero, el segundo campo del RelationField cuando el primero está vacio a pulsar en el se mostrará completo.
 Saltar Si el parámetro CampoASaltar esté relleno este campo por defecto está a true para que el campo se salte, si lo ponemos a false por defecto no se saltará pero tendrá el icono para activar y desactivar
 FieldListInMemory Lista de campos que se calculan en memoria, se admite el caracter "*"
EJEMPLO
[RelationFields] cd_prov, cd_muni
Define una dependencia del campo cd_muni respecto del campo cd_prov, la selección del subselect se define en un fichero "LDF" con el mismo nombre del campo quitando '"cd_" y acabado en "LDF" quedando en este ejemplo como " muni.ldf  ".
EJEMPLO
[RelationFields] cd_auto, cd_prov, cd_muni
Define una dependencia del campo cd_prov respecto del campo cd_auto, del campo cd_muni respecto del conjunto anterior, es decir del campo cd_prov que ya dependía del cd_auto. Por último forma una dependencia del campo cd_muni respecto de todos los anteriores en conjunto.

[RelationFields] sexo, cd_profe

[AddCode] a,A,?R | cd_profe | I | DynamicSQL=SqlProfesion

[PHPIni] a,A,?R// en multifichas va en el GDF
function SqlProfesion(){
global $_vF;
if( $_vF['sexo']<>'' ){
qQuery( 'select cd_profe,nm_profe_'.strtolower($_vF['sexo']).' from profe_sexo order by 2' );
}else{
qQuery( 'select cd_profe,nm_profe_h from profe_sexo order by 2' );
}
}

[JSIni] a,A,?R// en multifichas va en el GDF
function SqlProfesion(){
var Valor = '';
if( eGO('cd_profe').oValue!=undefined ) Valor = eGO('cd_profe').oValue;
if( eGF('sexo')!='' ){
return Array( 'select cd_profe,nm_profe_'+eGF('sexo').toLowerCase()+' from profe_sexo order by 2', Valor );
}else{

}
}
