DESCRIPCION
Código PHP a ejecutar después de dar un alta desde una ventana auxiliar y antes de cerrarla, lo que se genere está dentro de una zona javascript.
Es útil para asignar datos del registro que se acaba de dar de alta a la ventana padre.
Esta etiqueta está asociada a [WinClose] ya que sólo  se ejecuta cuando está esta etiqueta presente.
