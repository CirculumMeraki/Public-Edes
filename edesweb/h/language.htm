DESCRIPCION

Esta etiqueta se tienen que definir la primera y se utiliza para tener la aplicación en mas de un idioma. el DF donde se quiera poner el texto "TextoIdioma1" se pondrá "@VarLNG@", las variables tienen el mismo nombre que la primera columna con el caracter "@" antes y despues de la variable.
Hay una variable de sesión $_SESSION['_LANGUAGE_'] donde se mantiene la extensión del lenguage a visualizar.
Hay una variable global "@LNG@" para añadir el caracter "_" mas la extensión del lenguage.
y las funciones eLng(), y eLngLoad() 
y la variable $_Lng[$Cod], $_LngPublic['@'.$Cod.'@']
Desde el fichero "sql.ini" se puede activar una variable de configuración de [Language] para ver los textos que se han traducido y es "$_LanguageTron" que indicará el caracter de inicio y el final de todos los textos traducidos se suele poner el caracter Alt+126 "~".
Para utilizar el traductor "gsTranslater" es aconsejable utilizar la variable "_PROMP", "_PROMP2", "$_PROMP" o "$_PROMP2" para pasar variables de cadena a las funciones o pasar cadenas delimitadas por comillas.
Hay unas variables genéricas en  el top que son:
_xMESSAGE
_xCONFIRM
_xWARNING
_xERROR

_xACCEPT

_xNO
