[PagFoot] PieDePagina
DESCRIPCION
Establece el pié de página, por defecto está como "&b&bPág. &p/&P".
Para definir una configuración general se añadirá la variable en el fichero de configuración "sql.ini".
PARAMETROS
 PieDePagina &b El texto que está acontinuación se centra
&b&b El texto despues del primer "&b" se centra y el que está despues del siguiente "&b" se justifica a la derecha
&p Página actual
&P Total de páginas
&h Hora HH:MM
&D Date DD-MM-AAAA
&d Date MM/DD/AAAA
