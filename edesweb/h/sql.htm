$_CntMarginLeft: Distancia entre dos controles que se ponen seguidos cuando el segundo no tiene label.

Generales de DDBB 

$_Sql: Contiene, entre comillas simples, el nombre del controlador actual de la base de datos.
$_Limit: Número máximo de registros a extraer.

De Procesos

$_SqlHostName: Nombre del host del motor usado para procesos.
$_SqlUsuario: Nombre del usuario para el motor de procesos.
$_SqlPassword: Password para el usuario ($_SqlUsuario) del motor de procesos.
$_SqlDiccionario. Nombre del diccionario de datos a utilizar en el motor de procesos.

De Extracciones

$_SqlExHostName: Nombre del Host del motor usado para extracciones.
$_SqlExUsuario: Nombre del usuario para el motor de extracciones.
$_SqlExPassword: Password para el usuario ($_SqlExUsuario) del motor de extracciones.
$_SqlExDiccionario: Nombre del diccionario de datos utilizado en el motor de extracciones.

Otras

$_Estadística: Valor booleano (true/false ) que determina si se graban o no datos estadísticos de acceso, operaciones, etc.
$_Development: Valor booleano (true/false) que determina si el motor actual es de desarrollo o no.
$_Debug: Valor numérico que indica el modo de depuración actual:
 0. Sin depuración.
 2. Sentencias SQL.
 3. Muestra las líneas de las celdas.
