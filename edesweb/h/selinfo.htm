SelInfo
SINTAXIS
[SelInfo] Mode | Field |  SourceFile [ | TAB / T / LIST / L / NOEVENT ]
     TargetField = SourceField
     ...
[SelInfo] Mode | Field |  TargetField = SourceField [, TargetField = SourceField ]  | SourceFile [ | TAB / T / LIST / L / NOEVENT ]
...
[SelInfo] Mode | Field |  TargetField = SourceField [, TargetField = SourceField ]  : SourceFile
...
[SelInfo] Mode | Field |  NomFuncion()  : SourceFile
...
DESCRIPCION
Ayuda a seleccionar un dato en una ficha auxiliar o de listados mediante el código "B" en la columna modo de los campos, se buscará el registro metiendo condiciones en una ficha y seleccionándolo desde la lista.  Esta etiqueta puede ser multilinea pudiendo el tercer parametro ponerse en varias lineas incluida si se quiere en la linea de la etiqueta, con la siguiente sintaxis:

[SelInfo] Mode | Field |  TargetField = SourceField,...
TargetField = SourceTarget,...
[TargetField = SourceTarget ] : SourceFile 

[SelInfo] Mode | Field |
TargetField = SourceTarget,...
[TargetField = SourceTarget ] : SourceFile
PARAMETROS
TargetField: Campo del formulario destino, si algún campo de la lista se encuntra en una solapa distinta de la actual se tendrá que preceder el nombre del campo con el nombre del formulario punto campo ( ej: FRM1.campo ).
SourceField: Campo del formulario origen, este puede ser el nombre de un campo, el comando "onchange" o una función de usuario.
SourceFile: Fichero "fdf para generar la ficha o el formulario de selección, si se pone tiene que ir sin extensión.
El tercer parámetro, la lista de "TargetField = SourceField " puede sustituirse por una llamada a una función sin parámetros y los paréntesis sin espacio entre ellos, si este parámetro está en blanco buscará como fichero el nombre del campo con extensión "fdf".
EJEMPLO
[SelInfo] a,mR | _nm_empre | | cd_emp.fdf | TAB
     cd_empre = cd_empre
     _nm_empre = nm_empre
EJEMPLO
[SelInfo] a | cd_emp | cd_emp=cd_emp, _v_numero=numero, _v_dt_alta=dt_alta, _v_r_social=r_social : cd__emp 

[Fields]
    Empresa | cd_emp | CIF | T | 9 | | AB |||
 
EJEMPLO
[SelInfo] a | cd_emp | uSelEmpresa()

[Fields]
