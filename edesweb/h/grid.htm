SINTAXIS
[Grid] Field [ | [%][M][2][b]  [ | Definicion ] ]
...
DESCRIPCION
En un listado te hace el cruce entre varios campos. A continuación de la etiquéta se puede introducir el código del interior de la función para calcular la columna donde se colocará el dato, el parámetro que recibe es "$Value" y este código tiene que terminar con el comando "return" y el número de columna donde se colocará el dato.
PARAMETROS
EJEMPLO
[Grid] cd_propiedad
[Grid] cd_propiedad | % | select cd_propiedad, abreviatura from propiedad order by abreviatura
[Grid] cd_propiedad | | select cd_propiedad, nm_propiedad from propiedad where tipo='{Field}' order by cd_propiedad
[Grid] cd_propiedad | | A,AA; B,BB; C,CC
