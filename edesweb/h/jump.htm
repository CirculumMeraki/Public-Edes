[Jump] NomDF
DESCRIPCION
Salta a ejecutar otro fichero de definicion (DF). Una de sus utilizaciones es para cuando queremos poner una lupa de busqueda/selección en la ficha pondremos en la columna "modo" una "B" para buscar mediante una ficha ( poniendo condiciones ) o una "b" para seleccionar desde un listado, estas busquedas llamarán a los script "[NomCampo].fdf" y a "[NomCampo].ldf" respectibamente, estos script se utilizan para mostrar varios campos de las entidades a seleccionar pero cuando se puede hacer con el mismo script de la entidad en los DF "fdf" y "ldf" pondremos la etiqueta [Jump] para que salte a su script, es decir estará ejecutando otro script evitando duplicar el código.
Si no es un fichero eDes limpiará el buffer hará un include del fichero y ejecutará un exit.
