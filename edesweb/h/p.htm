SINTAXIS
{P} RefName [ | NoTD ]
DESCRIPCION
Es una sub-etiqueta de [Fields].
Añadir código PHP en una fila de la rejilla del formulario.
Dentro de la etiqueta [Fields], en la definición de un campo, como etiqueta insertamos "{P}" y el nombre de referencia (por ejemplo "myRef"). Después de la etiqueta [Fields] volvemos a insertar "[P]" seguido del mismo nombre de referencia y en las siguientes líneas el código PHP que necesitemos. Con éste método podemos instroducir código PHP a nuestro antojo en cualquier fila de la rejilla del formulario.
{P} captcha [ longitud=5 ]
Al mostrar el captcha se graba la variable de sesión $_SESSION["CAPTCHA"] para su posterior chequeo.
PARAMETROS
RefName: Nombre identificativo.
Si la constante "NoTD" no se pone te dejará una única celda con el cospan que corresponda para que metas lo que quieras, pero si le pones "NoTD" tu tienes que definir todos los TD de la fila.
EJEMPLO

[Fields]
   Client Code | cd_cli | ...
   {P} myRef
   ... 

[P] myRef
echo "This is a test";
 
EJEMPLO

{P} Captcha | 7
Pondrá un "capcha" en la linea actual con una longitud de 7 caracteres.
