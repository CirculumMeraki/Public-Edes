DESCRIPCION
Desactiva todas o algunas de las utilidades del listado. Si ponemos el nombre de una función de usuario esta se ejecutará al ejecutar las extracciones enviando la inicial de la acción a ejecutar y pudiendo devolver parámetros adicionales para la llamada por la URL al servidor.
Existe una función de usuario denominada "uFileSupport()" que si existe es llamada antes de descargar un listado en formato PDF, XLS, XML, MDB, TXT o CSV a esta función se le pasa el nombre del fichero, uno de los usos de esta función es que quede constancia de las extraciones que lo hará el desarrollador.
PARAMETROS
OpcionesADesactivar * Desactiva todas las utilidades
S Ordenanción ( Sort )
L Buscar datos ( Location )
P Imprimir ( Printer )
x Extraer a Excel ( XLS )
p Extraer a PDF ( PDF )
a Extraer a Access ( MDB )
m Extraer a XML
 t  Extaer a TXT
 v Extraer a CSV
C Chart
W Modo SubVentana
FuncionDeUsuario Función de usuario a ejecutar al pulsar el icono
UserFilterFunction Al filtrar datos pasará por la función de usuario indicada pasando el parámetro de lo que se va a buscar
ExtensionesAdicionales  Formato PDF, XLS, XML, TXT, MDB, CVS (aunque el usuario no lo tenga se podrá extraer en estos formatos).
Si se pone solamente la constante CMP en los lTools aparecerá un icono para comparar listados o informes, en este caso llevará un parámetro adicional NºColumnasIguales.
El parámetro CMP solo estará activo en informes que no sobrepase una página y no está presente la etiqueta [ColsOp] con "S" de grupos
NºColumnasIguales   Solo se define cuando el parámetro anterior es CMP e indica el nº de columnas que son iguales en la comparación (tendrán que estar al principio)
