Variables en PHP
Variables PHP:
 $_Mode Variable Modo
 $_SubMode Variable SubModo, solo en listados ( cl, ml, bl )
 $_Form Matriz con los datos de la etiqueta [Fields]
 $_pF Puntero a la matriz $_Form, eje: $_Form[ $pF["NomCampo"] ]
 $_HndDB Conexión con la base de datos
 $_InsertId Matriz con los valores seriales ( $_InsertId[NomTabla] )
 $_PSOURCE  Nombre del script que ejecutó la página ( WDESKTOP, WWORK ).
 $_Records Variable pública para almacenar el total de registros en un qCount()
 $_vF Valor de los campos tanto el las fichas como en los listados ( $_vF[ NomField ]  )
 $EmptyList Mostrará el listado aunque no tenga registros, por defecto es false.
 $_SourceScript Nombre del fichero de definición DF principal
 $_DownloadPDF Modifica el comportamiento de los listados, si esta variable vale "P" se descargará directamente un PDF y si vale "X" será un fichero Excel.
 $_ShowZero Valor numérico que indica como se visualizan los valores numéricos en las fichas ( la variable se insertará en [PHPStart] ), los valores son:
        1.- Se muestra el valor cero
        0.- Si es cero se muestra el campo vacio, valor por defecto.
       -1.- Se muestra cero solo si se mete el valor cero, es decir que en un campo numérico si pones cero se verá y si lo borras se verá vacio.
 $_ShowZeroFields Valor numérico que indica como se visualiza el valor numérico de un campo en una ficha, esta es una vaiable hash ( la variable se insertará en [PHPStart] ), los valores son:
        1.- Se muestra el valor cero
        0.- Si es cero se muestra el campo vacio, valor por defecto.
       -1.- Se muestra cero solo si se mete el valor cero, es decir que en un campo numérico si pones cero se verá y si lo borras se verá vacio.
 $_AutoJump Define si al escribir en un input al rellenar todos los carácteres se salta de campo automaticamente, si la variable es "true" será sobre todos los input, si se define una matriz hash se puede definir unos determinados campos.
$_AutoJump = true;
$_AutoJump = array( "cd_auto"=>1, ... );
$_AutoJump["cd_auto"] = true;
En "desktop5.css" esta la clase "AutoJump" con el color con el que se ejecuta el efecto a mostrar los campos de salto automático cuando se pone el foco.
$_SummaryNoHeaders Si esta variable la ponemos a "true" los informes con agrupaciones ([ColsOp] S) no mostrará la cabecera del grupo solo sacará los subtotales.
$_NOSELECTFILL Los campos select aunque no sean editables contienen todos los registros si se quiere, cuando el campos solo es de lectura, tiene dos formas de uso:
- $_NOSELECTFILL[ NmField ] = true;// anulas el relleno de los select de uno en uno
- $_NOSELECTFILL["*"] = true;     // anulas el relleno de todos los select
$_DBSEQUENCE Si estamos en el controlador oracle y queremos configurar que todas las tablas con secuenciales los generan trigger pondremos esta variable como "NO".

Etiquetas y variables:
Se tiene acceso a todas las etiquetas del lenguaje mediante la variable pública del mismo nombre en mayúsculas precedida de una barra baja, siendo por norma general una cadena si la etiqueta tiene solo un parámetro o una matriz si tiene mas de uno.

Variables de sesión:

 $_Node Código del nodo
 $_Tree Código del árbol de opciones
 $_User Código del usuario
 $_pxH Altura en pixel del monitor

Valores por defecto:
Estas variables van en la etiqueta [FIELDS] en la columna ocho valores por defecto.
 #clock# Formato "HH:MM:SS" dependiendo de la longitud del campo
 #default# En los modos (a,b,c,m) si en el desktop está almacenada la variable con el mismo nombre del campo tomará el valor por defecto.
 #sy2s# Formato ">YYYY-MM-DD 00:00:00" para búsquedas desde el inicio del día.
 #today# Formato "DD-MM-YYYY"
 #y2s# Formato "YYYY-MM-DD HH:MM:SS"

 

Variables en "sql.ini":
 $_DownloadUrl  URL para cuando las descargas se hacen desde otro Apache
 $_DownloadPath Directorio donde se guardarán los ficheros de las extracciones.
 $_DownloadDays Dias que se guardan las descargas.
$_TABFormType Define el tamaño de las fichas del grupo de fichas, estas puede ser:
Diferent: valor por defecto, cada ficha tendrá el tamaño en función de su contenido.
Ecual: todas las fichas tendrá el mismo tamaño, siendo tanto el ancho como el alto el mas desfavorable.
 $_NoContentTree Indica si al dar de alta o modificar usuarios los árboles que asignes tiene que estar dentro del tuyo, por defecto es false es decir sólo puedes asignar los árboles que te pertenezcan.
 $_CompressedPages Para desactivar la compresión de las páginas en Procesos.
 $_DEFAULTNOTOOLS Se indica las prestaciones que se desactivan en los listados, es como la etiqueta [NoTools] pero global.
 $_CheckBox $_CheckBox['H']['ON'] = ''; // HTM.True
$_CheckBox['H']['OFF'] = '';  // HTM.False
$_CheckBox['P']['ON']  = 'S';         // PDF.True
$_CheckBox['P']['OFF'] = '';           // PDF.False
$_CheckBox['X']['ON']  = 'S';         // XLS.True
$_CheckBox['X']['OFF'] = '';   // XLS.False.
 $_CheckBoxSave Valor de los checkbox al grabar, por defecto es array("S",""), matriz bidimensional con el valor true y el false.
 $_MarkVisited Clase para marcar las filas en los listados cuando se abre una subventana

$_QuestionsEmptyData = false;                      // Al preguntar en los select añade dos option "Sin datos" y "Con datos"
$_QuestionsEmptyDataColor = '#666666';     // Color de los option especiales

$_ExportExtras = false; // Añade dos opciones en los LTools: Exportar con Clave y Exportar por EMail
