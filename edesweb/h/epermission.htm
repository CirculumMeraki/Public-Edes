ePermission
SINTAXIS
ePermission( Label [, Script=OpDeEntrada ] )
DESCRIPCION
En el Desktop hay una opción para el mantenimiento de los "Permisos Especiales" que significa que dentro de una opción haya usuarios que puedan hacer alguna cosa distinta como por ejemplo poder modificar un dato delicado, esto se gestiona desde el usuario y a la hora de programar si se quiere buscar si tiene ese permiso especial se utiliza esta función, retorna un valor lógico.
Si la función se puede utilizar dentreo de las etiquetas [JS*] o [HTM*]
Cuando estas escribiendo en eDes también puede usar la forma "#P(Label)".
PARAMETROS
Label   Palabra clave o palabras separadas por coma en este último caso con cualquiera que encuentre devolvería true
Script Es opcional y por defecto será el script que se ha ejecutado la opción.
EJEMPLO
[JSEnd] mR
if( <?= ePermission('PuedeModificarDNI') ?> ) eEF('dni',false);
if( <?= !ePermission('PuedeModificarDNI') ?> ) eEF('dni',false);
El campo "dni" se manda editable y si el permiso "PuedeModificarDNI" no lo tiene pondrá al campo "dni" como no editable.
