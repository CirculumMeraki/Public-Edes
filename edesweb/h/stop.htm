DESCRIPCION
Mediante esta etiqueta se para el ciclo de las acciones de las fichas, por ejemplo si queremos que el usuario modifique su clave y una vez modificada queremos que se termine el proceso. Si se está ejecutando en una subventana esta se cerrará después del mensaje.
También se puede enviar la orden por url de la siguiente forma:

[JSEnd] *
top.eSWOpen( window, 'edes.php?FmR:tipopersonas.edf&_SEEK&cd_tipopersonas=1&_STOP=1' );
