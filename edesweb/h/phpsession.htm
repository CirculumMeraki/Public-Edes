PHPSession
SINTAXIS
[PHPSession] VariableCiente=VariableDeSesion, ... [ MensajeDeError ]
DESCRIPCION
Chequea en el lado del servidor que las variables de sesión que tenga el usuario asignadas sean las mismas que se manda en el POST y en el GET. Empareja cada variable de sesión con su variable POST o GET que esté presente en el script
PARAMETROS
VariableCiente  Las variables POST y GET 
VariableDeSesion Variables de sesión
MensajeDeError Por defecto es un mensaje al programador
EJEMPLO
[PHPSession] cd_auto=_Auto_, cd_prov=_Prov_, cd_coma=_Coma_, cd_muni=_Muni_
Si alguna variable de sesión de las indicadas está rellena tendrá que tener el mismo valor la variable de tipo POST o la de GET si no dará error de sesión. y no dejará realizar la operación.
