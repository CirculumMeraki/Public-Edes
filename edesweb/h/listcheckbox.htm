ListCheckBox
SINTAXIS
[ListCheckBox] Tipo | ON | OFF
DESCRIPCION
Define como se visualizan los "checkbox" en los listados. En el fichero de configuración "sql.ini" se configura la visualización general mediante la variables "$_CheckBox".
PARAMETROS
Tipo  Tipo de salida, se indicará la incial del formato de salida pudiendo ser: HTML, PDF o XLS ( H / P / X )
ON  Texto o imagen en caso de HTML cuando el control está activo.
OFF  Texto o imagen en caso de HTML cuando el control está desactivado
EJEMPLO
[ListCheckBox] H | <img src='g/tf_1.gif'>
[ListCheckBox] P | Yes | No
