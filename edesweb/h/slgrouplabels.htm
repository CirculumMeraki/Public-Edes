slGroupLabels
SINTAXIS
{slGroupLabels} LabelSubLista | LabelSubLista | ...
DESCRIPCION
Funciona igual que la etiqueta [LabelSubLista] pero para las sublistas, la peculiaridad es que las etiquetas {slSql} y {slMenu} ponemos una "#" con los datos a sustituir.
Tambi?n se puede poner de forma abreviada {slGL}.  En la columna de datos "slSql" no se puede poner el campo de la tabla padre.
En la columna de "slTH" se puede poner la barra "" para indicar lo que hay que mostrar seg?n el modo "a /mR cR/bR" ModoModificaci?nModoConsultas.
Las etiquetas contenidas en esta puede llevar o no el prefijo "sl" ej: "slAlign" o "Align"
