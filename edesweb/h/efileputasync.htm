eFilePutAsync
SINTAXIS
eFilePutAsync( FilePC, FileSrv [, Include, Variables, TextAlert  ] )
DESCRIPCION
Copia del PC al Servidor de forma Asincrona, sin limite de tamaño. Si al enviar ficheros desde un formulario está declarada la función "uFilePut( NmFile )" ejecutará la función de usuario que será la encargada de enviar el fichero.
PARAMETROS
FilePC  Nombre del fichero del PC
FileSrv  Nombre del fichero en el Servidor
Include  Php a ejecutar cuando se termine de enviar el fichero, llegan las variables:
$_NMFILE: Nombre del fichero grabado
$_User
$_Node
$_Tree
Variables Variables adicionales a enviar al fichero a ejecutar
TextAlert Texto a mostrar cuando se termine de enviar el fichero
EJEMPLO
top.eFilePutAsync( FilePC, '/_tmp/zip/'+_User+'_'+Date.parse(new Date()), 'asignar.php', '$_MD5="'+eGF('md5')+'";', 'Fichero "'+eGF('nm_gs_img')+'" grabado' );
