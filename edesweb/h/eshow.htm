eShow
SINTAXIS
eShow( Campos [ [ , Nivel/TagName ] , OcultaElEspacio=true ] )
DESCRIPCION
Visualiza el "Campo" o el "parentElement" de nivel "Nivel" si es un valor numérico o hasta que encuentre el "TagName" de sus "parentElement" sucesivos, el parámetro nivel puede tener un valor especial "GROUP" para mostrar un grupo de datos comprendidos entre varias lineas divisorias y "FIELDSET" que mostraá la linea donde se encuentra el "FIELDSET"., también si el parámetro es "-" mostrará hasta el <HR> y si ponemos "L" ocultará su label asociado.
PARAMETROS
 Campos  Nombre del campo o lista de campos separados por comas 
 Nivel/TagName  Nivel del elemento padre que se quiere visualizar o el TagName del padre que se quiere visualizar (opcional, por defecto es "0")
- Un Nº: asciende nº parentNode y lo oculta
- TAG: asciende hasta el tag indicado y lo oculta.
- "GROUP":
- "FIELDSET":
- "-": oculta hasta el <HR>
- "L": oculta el label asociado.
- "submit": Bot?n de submit
- "slSubmit": bot?n de submit de SubList
 OcultaElEspacio  La opción por defecto es ocultar eliminando el espacio, pero si lo ponemos a false ocultará el elemento dejando intacto el espacio que ocupa
EJEMPLO
eShow( 'ccc' );                 // Visualiza el campo "ccc"
eShow( 'ccc', 2 );              // Visualiza "ccc.parentElement.parentElement";
eShow( 'ccc', 'TR' );           // Visualiza el elemento padre "TR" del campo "ccc"
eShow( 'banco,sucur,dc,ccc' );  // Visualiza los cuatro campos
eShow( 'ccc', '-' ); // Visualiza hasta la linea
eShow( 'ccc', 'L' ); // Visualiza el campo y su label
eShow( 'submit' );   // Visualiza el botón del submit
eShow( 'submit','tr' );  // Visualiza el TR donde está el submit
