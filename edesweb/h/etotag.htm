eToTag
SINTAXIS
eToTag( Objeto, Etiqueta [ [, NVeces=1 ] , TagsDePartida ] )
eToTag( Objeto, Atributo, Valor [, TagsDePartida ] )
DESCRIPCION
Devuelve el Objeto padre cuya etiqueta es "Etiqueta", si el tagname del objeto actual es el mismo que el de la etiqueta no hará nada a no ser que el "NVeces" sea menor de cero, en cuyo caso lo hará ese número de veces.
PARAMETROS
Objeto  Objeto hijo
Etiqueta TagName del Objeto que queremos que nos devuelva  o varios separados por coma.
Puede contener también el nombre de un atributo.
NVeces Cuando por ejemplo el evento se genera en un td y tu quieres cojer el objeto table del table de ese td pondremos 2 porque queremos no el primer table si no el segundo.
Puede contener también el valor de un atributo.
TagsDePartida Parámetro opcional es una lista de tags separados por coma para indicar desde cuales se calcula el padre
EJEMPLO
var Obj = top.eToTag( event.srcElement, 'table' );
Te devolvera el Objeto TABLE
EJEMPLO
var Obj = top.eToTag(event.srcElement,'TR',1,'IMG,TD'), n, css;
if( Obj==null ) return;
Si el evento no parte de un "TD" o un "IMG" devolverá NULL en caso contrario te devolverá el objeto "TR"
EJEMPLO
var Obj = top.eToTag( event.srcElement, 'className', 'Resaltado' );
Devuelve el primer objeto padre que su "class" sea "Resaltado";
EJEMPLO
var Obj = top.eToTag( event.srcElement, 'TABLE,TR' );
Primero ascenderá buscando la primera "TABLE" y luego continuará hasta encontrar a "TR".
