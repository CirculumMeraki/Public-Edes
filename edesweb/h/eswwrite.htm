eSWWrite
SINTAXIS
eSWWrite( html )
DESCRIPCION
Graba codigo HTML en la proxima ventana que se cree al abrirla con src igual a null.
PARAMETROS
EJEMPLO
top.eSWWrite( "<form action='edes.php?FA:script.edf' METHOD='POST'>" );
top.eSWWrite( "<input name='campo1' value='99'>" );
top.eSWWrite( "<input name='campo2' value='XXX'>" );
top.eSWWrite( "</form>" );
top.eSWOpen( window, null, 'Alta', 1, 500,500 );
