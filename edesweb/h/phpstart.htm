PHPStart
SINTAXIS
[PHPStart] Mode [ [ | NomDF,.../else ] | UNIQUE/Condition ]
...
DESCRIPCION
PHP a ejecutar en el inicio, esta etiqueta se debe poner la primeta en el DF.
Si se quiere acceder a base de datos hay que llamar a la función "eInclude( $_Sql );"
PARAMETROS
El parámetro "UNIQUE" es para incluir esta zona de código una sola vez en caso de tener funciones declaradas y halla conflicto con los modos.
EJEMPLO
[PHPStart] M
eUnSet('re_num_devoluciones');
En este caso dejamos visualizar el dato nº de devoluciones pero al modificar lo eliminamos por ser un campo que sólo queremos que lo pueda modificar el sistema..
