FormStatic
SINTAXIS
[FormStatic]
[FormStatic] NomSubLista | CampoDestino=CampoOrigen, ... |  Operaciones en Fila | Operaciones en Columna | NomFuncUsu | CódigoHTML
DESCRIPCION
Esta etiqueta tiene dos comportamientos dependiendo de si tiene parámetros o no:
Sin parámetros: Solo en modo alta te permite dar altas sin enviar la ficha al servidor, mediante este sistema solo viajan los datos obteniendo mas velocidad, actualmente no soportado en el grupo de fichas.
Si existe la función de usuario con el nombre "FUNCTION_FormStatic()" será ejecutada, que servirá para ejecutar alguna acción no automática.
 Cada vez que se dá un alta todos los campos se limpian y los que tubieran algún valor por defecto se refrescan, si quisieramos que algún campo no se limpie deberemos indicarlo mediante la etiqueta:
[AddCode] a | ...NomCampo... | i | NoClear
Con parámetros: Identifica que este EDF realiza el mantenimiento de una SubLista sin ir al servidor en ninguna de sus operaciones, será el formulario donde esté la SubLista quién mandará los datos al servidor, el [DBIndex] de este EDF tiene que ser el campo de relación con la tabla padre donde esta la [SubList]. No se permiten campos del tipo "file".
Las etiquetas [JSEnd], [HTMEnd], [PHPEnd] y [MsgAnswer] no está disponibles en [FormStatic]
PARAMETROS
 NomSubLista El nombre de la SubLista tiene que ir encerrado entre corchetes
 Igualdad Igualdad ( CampoDestino=CampoOrigen) Nombre del campo de la SubLista = Nombre del campo de la ficha, si los dos campos se llaman igual solo se pondrá uno y si se quiere coger el innerText de un campo select pondremos un "*" delante del nombre del campo de origen.
 OperacionesEnFila Parámetro no implementado.
 OperacionesEnColumna Las operaciones son: (+) para sumar, (c) para contar las filas y (#) para contar las filas que estén rellenas.
La sintaxis es:  NomCampoDestipo=[+/c/#], ....
 NomFuncUsu Nombre de la función de usuario, sin parámetros, se pasa el objeto TR que se ha modificado o insertado, si se ha borrado el TR se pasará NULL.
 CódigoHTML Código HTML a insertar en la columna marcada como "IMG" se utiliza para una columna del listado meter iconos con las opciones de Alta, Baja y Modificación. 
EJEMPLO
[FormStatic] [_clau] | ''=IMG, cd_tclausula, nm_contra_clau | | ,c | CalculaValores | ...
