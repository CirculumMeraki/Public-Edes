9. Condición del campo
DESCRIPCION

Marca una condición sobre el campo.

- Si ponemos sólo un "#" se obliga a meter el campo. 
- Si ponemos sólo un "=" se obliga a rellenarlo completamente.
- Si ponemos sólo un "%" el campo tendrá que estar vacío o completamente relleno.
- Si ponemos una condición, donde encuentre el "#" lo sustituirá por el valor actual del campo. Ejemplo: "# > 100" ( si queremos poner 'or' no podremos utilizar la forma "||" ) .
- Por defecto las condiciones actuan sobre los modos "a" y "mR" pero si se quiere que actue sobre los modos "c", "m" o "b" se separaría las condiciones con el carácter "" quedando la condición normal a la izquierda y a la derecha la condición para los modos  "c", "m" o "b".

El tipo de condición "=" o "%" por defecto en los listados centrarán los datos en sus columnas.
 
Existen otras formas de marcar condiciones, como son las etiquetas [JSCheck], [FormCkeck].

El campo tipo "Periodo" se puede condicionar de la siguiente forma:
Con "#" indicamos que es obligatorio pero si a la vez queremos definir un rango de periodos le pondremos el carácter "/" y a continuación el rango de periodos, este se puede definir de la siguiente forma:
[Desde] [,[Hasta] [, [MesesSeleccionables] ] ]
Tanto "Desde" como "Hasta" puede ser:
- "": Cadena vacía para indicar sin límite en este parámetro
- "AAAA": Indica el año
- "AAAA-MM": identifica el periodo
- FunctionUser(): hace la llamada a una función de usuario que deverá devolver uno de los valores anteriores obteniendo el valor en la posición que se encuentra excepto que si solo hay una unción si devuelve una cadena será del parámetro "Desde" y si devuelve un array se obtendrán los tres parámetros.
- El año se puede sustituir por "YEAR" y el mes por "MONTH".
- A la vez podremos sumar o restar numeros de años o meses poniendolo entre parentesis ej: "(YEAR-5)-01", "(YEAR-5)-(MONTH+2)", etc.
- También se puede pasar dos periodos separados por una "/" tomando el valor mayor en caso de "Desde" y el menor en caso de "Hasta" como: "(YEAR-5)/2007".
- El dato "MesesSeleccionables" es una lista de nº de meses seleccionables separados por "," por defecto son todos, ej: "01,04,07,10"

[PHPIni] ?,a,mR || unique
function PeriodoHasta(){
   return '2014-02';
}
function Meses(){
   return '1,4,7,10';
}
function Todo(){
   return array( 2003, '2015-12', '2,4,6,8,10' );
}

En campo tipo "Fecha" se define igual que el tipo "Periodo" pero se añade la constante "DAY" y el tercer parámetro es una lista de nº de día de la semana seleccionable, por defecto son todos, del 0 al 6 siendo el 0 el lunes. En el campo fecha hay que definir las fechas completas.
