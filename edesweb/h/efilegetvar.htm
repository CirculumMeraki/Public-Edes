eFileGetVar
SINTAXIS
eFileGetVar( NomGrupo[.NomVar]  [ [,var1 ] , var2 [... var10] )
DESCRIPCION
Recupera una o todas las variables del grupo de un fichero de variables llamada "/_datos/config/group.var".
eFileGetVar( 'LogHistory', true ) -> Hace globales todas las variables y si solo hay una la devuelve.
eFileGetVar( 'LogHistory' , $array) -> Devuelve la variable "$array" como un array hast relleno con todas las variables dentro de "NomGrupo".
eFileGetVar( 'LogHistory', $v1, $v2, ... ) -> Devuelve los valores en los parámetros ( var locales ).
eFileGetVar( 'LogHistory.var2' ) -> Devuelve el valor de la variable "var2".
eFileGetVar( '/_datos/config/desktop.ini->$_Language' ) -> Devuelve el valor de la variable "$_Language" del fichero "'/_datos/config/desktop.ini".
eFileGetVar() -> Devuelve una matriz de matrices con todas las variables
La estructura del fichero "group.var" cada grupo se define con una palabra clave finalizada en ":" y en cada linea el nombre de una variables, el signo igual y su valor, no hace falta delimitar los valores entre comillas, aunque se puede poner y tampoco hace falta finalizar con el signo ";", si una variable de cadena su valor acabará en ";" habría que encerrar entre comillas todo su valor porque si no se eliminaría, cada linea de variables se puede sangrar no afecta.
El fichero "group.var" tiene la siguiente estructura:

Grupo1:
   key1 = Var1
   key2 = Var2
   ...
Grupo2:
   ...
