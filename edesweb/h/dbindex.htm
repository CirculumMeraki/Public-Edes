SINTAXIS
[DBIndex] FieldsList [[; FieldsList [; FieldsList [ | Edit [ | CheckIndex [ | ShowFields [ | Message [ |  FuncUser [ | DDBB ] ] ] ] ] ] ] ]
DESCRIPCION
Sirve para seleccionar los índices únicos de la tabla elegida en [DBTable]
El primer FieldsList es el índice único principal de actualización y los siguientes opcionales son los indices únicos secundarios.
Si no se define y la tabla tiene un solo registro se podrá modificar y consultar directamente con "#mR:script" o "#cR:script"
PARAMETROS
FieldsList  Lista de campos separados por comas, si ponemos ";" podremos poner una lista de campos (indice único) que no se puede repetir, este segundo indice si solo tiene un campo no hará el chequeo si el campo está vacio.
Edit Esta constante es para poder editar el campo indice cuando tiene un solo campo el indice y solo en fichas.
CheckIndex Constante "true" o "t" para indicar que se chequeen los campos del indice despues de introducir los valores, antes de rellenar toda la ficha. Con este flag activado si vamos a meter un dato que ya existe nos mostrará el mensaje "El registro yá existe."
Solo está activo con un indice con uno o mas segmentos y si tiene dos indices y uno es un serial.
ShowFields Lista de campos a mostrar cuando el registro que se quiere dar de alta ya existe (opcional).
Message Mensaje a mostrar si el registro ya existe (opcional), el Mensaje al ponerse como título normalmente va acompañado de una lista de campos a mostrar definidos en ShowFields.
En este parámetro se puede definir a la vez el Título y SubTítulo de la ventana separando los textos por una barra invertida.
Puede llevar etiquetas html.
FuncUser Nombre de función de usuario a llamar si el registro ya existe.
DDBB Nombre del fichero de definición de DDBB para base de datos alternativas.
EJEMPLO
[DBIndex]  cd_obra; cd_obra2 || True | cd_obra2, nm_obra | LA OBRA YA ESTA DADA DE ALTA
