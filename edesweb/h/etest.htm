eTest
SINTAXIS
top.eTest( window, NomFuncion, ArrayDeParametros [, NomFuncion, ArrayDeParametros ] ... )
eTest( NomFuncion, ArrayDeParametros [, NomFuncion, ArrayDeParametros ] ... )
DESCRIPCION
Función en el top para realizar pruebas unitarias, al ejecutarlo se verá el resultado en una subventana mostrando en rojo los resultados que no coinciden.
PARAMETROS
 window window (constante)
 NomFuncion Nombre de la función a ejecutar 
ArrayDeParametros  Es un array multidimensional, el último valor es el resultado correcto y los primeros son los valores a pasar a la función o los campos a asignar valores, en este caso serán los primeros valores a pasar en la matriz indicandolo de la forma "campo=valor".
El "resultado correcto" puede ser: un número, una cadena, un valor lógico, una expresión regular, null o undefined.
EJEMPLO
[JSEnd] a
eTest('S.date', [
     ["Y-m-d", 1, 2, 1998, "1998-02-01"]
]);

eTest('eOkDate', [
     ['10', '10-03-2016'],
   ['11',' 10-03-2016'],
   ['1003', '10-03-2016']
]);

eTest('Prueba', [
          ['campo1=20', 'campo2=30', 600],
          ['campo1=30', 'campo2=40', 1200],
          ['campo1=20', 'campo2=30', 200]
     ],
     'Prueba2', [
          ['campo1=20', 'campo2=30', 10, 610],
          ['campo1=30', 'campo2=40', 20, 1220],
          ['campo1=20', 'campo2=30', 30, 230]
     ],
     'Prueba3', [
          ['campo1=20', 30, 10, 610],
          ['campo1=30', 40, 20, 1220]
     });
